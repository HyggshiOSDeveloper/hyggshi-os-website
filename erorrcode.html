Tra cứu Mã Lỗi

Tìm kiếm mã lỗi hoặc từ khóa để xem cách khắc phục.

0x0001214 ⚠️

Lỗi kết nối API

"No response text found" / "Connection Failed"

Nguyên nhân: Lỗi mạng hoặc GitHub API rate limit.

Khắc phục: Kiểm tra mạng và thử lại sau 5 phút.

SYS_BOOT_ERR 🚫

Lỗi Khởi động

Treo logo hoặc màn hình đen khi bật.

Nguyên nhân: Xung đột script hoặc thiếu assets.

Khắc phục: Cài đặt lại Hyggshi OS Core.
